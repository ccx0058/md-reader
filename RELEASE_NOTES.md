# MD Reader v1.0.0 发布说明

## 🎉 首个正式版本

这是 MD Reader 的首个正式版本，一个轻量级、功能丰富的 Markdown 阅读器。

## ✨ 主要特性

### 核心功能
- 📝 支持 Markdown、纯文本、HTML 文件
- 🎨 15+ 精美主题（GitHub、Typora、VS Code、Dracula 等）
- 🌓 亮色/暗色模式自动切换
- 🔍 强大搜索功能（正则表达式、大小写敏感、全词匹配）
- 📑 智能大纲自动生成
- 🌲 文件树浏览

### 渲染能力
- 💻 多语言代码高亮
- 🎨 括号彩虹着色
- 📐 KaTeX 数学公式
- 📊 Mermaid 图表支持
- 🖼️ 图片点击放大
- 🔗 文档内链接跳转

### 实用工具
- 🔖 书签管理
- 📜 最近文件列表
- 📊 字数统计（字数、字符、行数）
- 💾 阅读进度自动保存
- 🖨️ 打印和导出（PDF/HTML）
- ⌨️ 丰富的快捷键

## 📥 下载安装

### Windows 用户
1. 下载 `md-reader-windows-amd64.exe`
2. 双击运行即可使用
3. （可选）运行 `register-file-association.ps1` 设置 .md 文件关联

### 系统要求
- Windows 10/11
- WebView2 运行时（Windows 11 自带，Windows 10 可能需要安装）

## 🚀 快速开始

### 打开文件
- 点击"打开文件"按钮
- 拖放文件到窗口
- 使用快捷键 `Ctrl+O`
- 双击 .md 文件（需配置文件关联）

### 常用快捷键
- `Ctrl+O` - 打开文件
- `Ctrl+Shift+O` - 打开文件夹
- `Ctrl+F` - 搜索
- `Ctrl+P` - 打印
- `Ctrl+=` / `Ctrl+-` - 调整字体大小
- `F11` - 全屏
